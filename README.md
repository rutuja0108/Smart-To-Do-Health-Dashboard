# 🚀 Smart To-Do Health
### *Engineering Productivity with Wellness Logic*

![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg) 
![JS](https://img.shields.io/badge/Vanilla-JavaScript-yellow.svg) 
![CSS](https://img.shields.io/badge/UI-Glassmorphism-purple.svg)

**Smart To-Do Health** is a sophisticated productivity application designed to bridge the gap between daily task management and mental well-being. Built with a focus on clean UI and functional logic, it uses a **dynamic health algorithm** to calculate a user's "Vitality Score" based on real-time task completion ratios.

---

## ✨ Key Features

* **📊 Automated Health Analysis:** A custom-coded logic engine that monitors your **Vitality Score (0–100%)**. As you conquer tasks, your health status evolves dynamically from *Building Momentum* to *Peak Performance*.
* **🩺 Integrated Dashboard:** A high-level view of your daily output, featuring real-time progress bars and automated task distribution stats.
* **🎨 Aesthetic Theme Palette:** Includes a built-in theme switcher (Midnight, Sakura, Emerald) that utilizes CSS Custom Properties for instant UI updates.
* **💾 LocalStorage Persistence:** Leveraging the **Web Storage API**, all tasks, health trends, and theme preferences persist across browser sessions without a backend.
* **📱 Responsive Glassmorphism:** A premium "frosted glass" interface designed with CSS Backdrop-filters, ensuring a seamless experience across all devices.

## 🧠 The Vitality Algorithm
The application evaluates user health by calculating the ratio of productivity to active workload:

$$VitalityScore = \left( \frac{\text{Completed Tasks}}{\text{Total Tasks}} \right) \times 100$$

* **90% - 100%:** 🚀 *Peak Performance* (Dopamine-optimized feedback)
* **50% - 89%:** 🌊 *In the Flow*
* **1% - 49%:** 🌱 *Building Momentum*
* **0%:** 💤 *Resting / Awaiting Input*
